Overview
- Personal website of Parv Shah built with Next.js 15 and React 19.
- Clean, minimal UI with a compact card layout.

Getting Started
- Prereqs: Node 18.18+ and npm/yarn/pnpm
- Install deps: `npm install`
- Dev: `npm run dev`
- Build: `npm run build`
- Start: `npm start`

Environment Variables
- Copy `.env.example` to `.env.local` and fill values:
  - `NEXT_PUBLIC_EMAILJS_SERVICE_ID`
  - `NEXT_PUBLIC_EMAILJS_TEMPLATE_ID`
  - `NEXT_PUBLIC_EMAILJS_PUBLIC_KEY`
